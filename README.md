# VirtualSlime-API

## Usage

### Install Dependencies

```bash
yarn install
```

### Test dev (localhost:3001)

```bash
yarn dev
```

## Use Product

### Build

```bash
yarn build
```

### Start

```bash
NODE_ENV=production yarn start
```

## API Documents

- /api/posts

```
GET api.v-sli.me/api/posts?q=<Query>
```

`?q` : Search text : `string+string+string....`
